# HW2 - Meeting Notes to Action Items

A simple GenAI workflow that converts raw meeting notes into structured action items using an LLM.

## Business Workflow
- **Workflow:** Meeting Notes → Action Items
- **User:** Anyone who runs or attends meetings and needs a clean follow-up summary
- **Input:** Raw, unstructured meeting notes
- **Output:** A structured list of action items with owner and deadline where possible
- **Why automate:** Saves time after every meeting, reduces missed follow-ups, creates consistent formatting
